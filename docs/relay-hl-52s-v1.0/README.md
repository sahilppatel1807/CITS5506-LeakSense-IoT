# SRD-05VDC-SL-C / HL-52S V1.0 说明

## 这是什么

你手上的 `HL-52S V1.0` 是一个常见的 **5V 单路继电器模块**。  
模块上的蓝色大方块 `SRD-05VDC-SL-C` 是继电器本体，板子上通常还带有三极管驱动、续流二极管，有些版本还带光耦。

它的作用是：

- 用单片机的小电流控制信号
- 去开关一个更高电压/更大电流的回路

最常见用途：

- 控制风扇
- 控制小水泵
- 控制灯
- 控制蜂鸣器、电磁阀等 5V/12V 负载

## 继电器本体参数

`SRD-05VDC-SL-C` 常见关键参数：

- 线圈电压：`5V`
- 线圈功耗：约 `0.36W`
- 线圈电流：约 `70mA`
- 触点形式：`1C`，也就是 `COM / NO / NC`
- 常见触点额定值：`10A 250VAC` 或 `10A 30VDC`

注意：

- 这是 **触点额定值**，不是说你随便接到板子上就能安全跑到这个电流
- 实际安全能力还取决于模块 PCB 走线、端子、线材、绝缘距离和你的接线方式

## 引脚怎么认

### 控制侧

不同 HL-52S 板子丝印可能不完全一样，常见有这些写法：

- `VCC / GND / IN`
- `DC+ / DC- / IN`
- `+ / - / S`

通常对应关系是：

- `VCC` 或 `DC+` 或 `+`：模块电源，接 `5V`
- `GND` 或 `DC-` 或 `-`：地
- `IN` 或 `S`：控制输入

### 负载侧

螺丝端子一般是这三个：

- `COM`：公共端
- `NO`：Normally Open，常开
- `NC`：Normally Closed，常闭

它们的逻辑是：

- 继电器**不吸合**时：`COM` 和 `NC` 导通
- 继电器**吸合**时：`COM` 和 `NO` 导通

所以：

- 你希望默认断开，用 `COM + NO`
- 你希望默认接通，用 `COM + NC`

## 怎么接到 ESP32 / Arduino

### 推荐接法

- 模块 `VCC` 接开发板 `5V`
- 模块 `GND` 接开发板 `GND`
- 模块 `IN` 接任意数字 GPIO

不要这样接：

- 不要把继电器线圈直接接 GPIO
- 不要让 GPIO 给继电器线圈供电

原因是线圈电流通常在几十毫安量级，GPIO 扛不住。

### ESP32 能不能直接驱动 IN

大多数这类模块在 `5V` 供电时，`IN` 用 `3.3V` 的 ESP32 GPIO 也能驱动。  
但很多板子是 **低电平触发**，也就是：

- `GPIO = LOW`：继电器吸合
- `GPIO = HIGH`：继电器释放

也有少数是高电平触发，或者可切换高低电平触发。  
最稳的办法是上电后先做一次简单程序测试。

## 最简单的工作方式

假设你要控制一个 5V 风扇，默认关闭：

1. 电源正极接 `COM`
2. `NO` 接负载正极
3. 负载负极回到电源负极
4. ESP32 GPIO 控制 `IN`

这样继电器吸合时，`COM` 与 `NO` 接通，负载上电。

## 一个非常简单的测试程序

如果模块是低电平触发，这段常常能直接工作：

```cpp
constexpr int kRelayPin = 19;

void setup() {
  pinMode(kRelayPin, OUTPUT);
  digitalWrite(kRelayPin, HIGH);  // 默认释放
}

void loop() {
  digitalWrite(kRelayPin, LOW);   // 吸合
  delay(1000);
  digitalWrite(kRelayPin, HIGH);  // 释放
  delay(1000);
}
```

如果你发现逻辑反了，就把 `HIGH/LOW` 对调。

## 使用时最重要的注意事项

### 1. 模块电源一般要 5V

`SRD-05VDC-SL-C` 是 5V 线圈继电器。  
所以模块电源通常应该接 `5V`，不是 `3.3V`。

### 2. 单片机侧和负载侧是两件事

控制侧：

- `5V`
- `GND`
- `IN`

负载侧：

- `COM`
- `NO`
- `NC`

不要把这两边混在一起看。

### 3. 如果接市电，要非常谨慎

如果你打算切换 `240VAC` 市电：

- 一定要断电接线
- 一定要做好绝缘
- 端子不能裸露
- 线径和端子压接要合适
- 学生项目阶段尽量先用低压直流负载做实验

如果只是做课程项目验证，优先建议你先控制：

- 5V 风扇
- 12V 小灯
- 小水泵

不要一上来就碰市电。

## 结论

如果你只是想把它接到 ESP32 上先跑起来，优先按这个思路：

- `VCC -> 5V`
- `GND -> GND`
- `IN -> 一个 GPIO`
- 负载接 `COM + NO`

然后先用一个简单的开关测试程序，看它到底是高电平触发还是低电平触发。
