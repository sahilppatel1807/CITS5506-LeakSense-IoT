Getting Started

Last revision 2026/01/31

This article guides you through setting up the Arduino IDE for FireBeetle 2 ESP32-E, including downloading the software, configuring preferences, and installing necessary SDKs and boards.

Download Arduino IDE

  1. Click to enter the official Arduino website

  2. Select and download the Arduino IDE package that is appropriate for your computer OS.

Configure Arduino IDE

It is required to install ESP32 board in Arduino IDE for using FireBeetle 2 ESP32-E for the first time.

  1. Open Arduino IDE, click File-Preferences, as shown below

  2. Click the icon circled in red in the Preferences window.

  3. Add the json URL below to the newly-popped up window. There should be one URL for each line.
    https://espressif.github.io/arduino-esp32/package_esp32_index.json

  4. Click OK then.

  5. Download SDK. Click Tools->Board>Boards Manager.

  6. Enter "ESP32" in the search bar, then the SDK for esp32 will appear automatically. Click install now.

  7. When the SDK installation completes, the word "installed" will appear at the position marked in red below. Close the window.

  8. Select FireBeetle 2 ESP32-E development board. Click Tools-->Board-->ESP32 Arduino-->FireBeetle ESP32-E, as shown below.

  9. After that, connect your FireBeetle 2 ESP32-E to your computer. Here, a port COM4 comes out after the connection, indicating that it is the port for FireBeetle 2 ESP32-E. Select it as the way shown below.

  10. Now, the FireBeetle 2 ESP32-E is configured in Arduino IDE and it's ready to go.
